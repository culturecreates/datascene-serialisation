# Les balises HTML autorisées

*Les balises HTML autorisées b, i, u, sup et sub.*

## Examples

  ```json
  "<b>Texte en gras</b>"
  ```

  ```json
  "<i>Texte en italique</i>"
  ```

  ```json
  "<u>Texte soulign\u00e9</u>"
  ```

  ```json
  "<sup>Texte en exposant</sup>"
  ```

  ```json
  "<sub>Texte en indice</sub>"
  ```
